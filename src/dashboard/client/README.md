# src/dashboard/client/

Single-page app modules — browser-side React-like state management and UI logic. Runs in the user's browser.

## Files

- `campaigns.js` — Campaign list, create/edit/delete, drill-down views
- `calls.js` — Call history, playback, transcript view, outcome details
- `analytics.js` — KPI charts, trends, call funnel, per-employee metrics
- `prompts.js` — Prompt management UI, template editor, preview
- `brochures.js` — Brochure upload, preview, WhatsApp send history
- `settings.js` — User preferences, API key rotation, team invites
- `state.js` — State machine (current view, selected campaign, filters, etc.)
- `utils.js` — API client, fetch helpers, error handling, formatters
